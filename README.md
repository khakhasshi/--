# 塔罗牌抽牌应用

这是一个简单的网页应用，用于模拟塔罗牌抽牌过程。

## 功能
- 包含完整的78张塔罗牌（22张大阿卡纳 + 56张小阿卡纳）。
- 支持自定义抽取数量（1-78张）。
- 带有洗牌动画和翻牌效果。
- 随机出现逆位牌（Reversed）。

## 使用方法
1. 双击打开 `index.html` 文件。
2. 在输入框中输入想要抽取的牌数（默认为3张）。
3. 点击“洗牌并抽取”按钮。
4. 等待洗牌动画结束，牌面将自动展示。

## 文件结构
- `index.html`: 页面结构
- `style.css`: 样式与动画
- `script.js`: 逻辑控制
